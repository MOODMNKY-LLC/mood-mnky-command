# Media Library

Upload, browse, and manage digital assets. Images are used for fragrance scenes, product photos, and brand materials.

## Buckets

Choose a bucket when uploading:

| Bucket | Use |
|--------|-----|
| **Product Images** | Shopify product photos |
| **AI Generations** | AI-generated fragrance scenes |
| **Brand Assets** | Logos, mascots, videos, PDFs |
| **User Avatars** | Profile pictures |
| **Private Documents** | Internal files (not public) |

## Uploading

1. Select a **bucket** from the dropdown
2. **Drag and drop** files or click to select
3. Files are validated by size and type per bucket
4. Assets appear in the grid with thumbnail, name, and metadata

## Browsing and Filtering

- **Bucket filter** – All or specific bucket
- **Category filter** – Filter by category when set
- **Search** – Search by filename or tags
- **View mode** – Grid or list

## Asset Actions

- **Copy URL** – Copy the public URL
- **Sync to Notion** – Push image URL to linked fragrance's Notion page
- **Edit tags** – Add or remove tags
- **Delete** – Remove the asset

## Assigning to Fragrances

1. Select an asset
2. Use **Assign to fragrance** to link it
3. Click **Sync to Notion** to update the Notion page

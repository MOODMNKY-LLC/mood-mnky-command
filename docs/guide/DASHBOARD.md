# Dashboard

The Dashboard is your home base. It shows an overview of your lab and quick access to key features.

## Stat Cards

Four cards display counts:

| Card | Description |
|------|-------------|
| **Formulas** | Bath & body formulas from Whole Elise catalog |
| **Fragrance Oils** | Synced from Notion (or "Connect Notion" if not synced) |
| **Products** | Active on Shopify (or "Connect Shopify" if not connected) |
| **Collections** | On Shopify |

## Activity Feed

Shows recent activity such as:

- Formula syncs
- Fragrance oil syncs
- Product pushes to Shopify

## Quick Actions

Four shortcuts:

- **New Formula** – Go to Formula Catalog
- **Browse Oils** – Go to Fragrance Oils
- **Build Product** – Go to Product Builder
- **Blending Lab** – Go to Fragrance Blending

## Shopify Status

- Shows whether Shopify is connected
- Link to manage connection
- Shop name and domain when connected

## Notion Status

- Shows whether Notion sync is configured
- Last sync time for fragrance oils
- Link to Notion Sync page

# Glossary (Fragrance Notes)

The Glossary lists fragrance notes from the MNKY Note Glossary. Use it to explore notes and see which oils contain them.

## Browsing Notes

1. Go to **Glossary** in the sidebar
2. Use **letter filter** – Click A–Z to filter notes by first letter
3. Use the **search** box to find notes by name
4. Click a note to open the detail sheet

## Note Detail

When you select a note:

- **Definition** – Description of the note
- **Related Oils** – Fragrance oils that contain this note (top, middle, or base)
- **Family** – Fragrance family when applicable

## Tips

- Use the glossary to discover oils by note (e.g., find all oils with "Bergamot")
- Related oils link to the Fragrance Oils page for quick navigation

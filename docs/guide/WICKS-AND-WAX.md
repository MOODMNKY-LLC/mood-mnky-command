# Wicks & Wax Reference

Wick sizing guide, wax type compatibility, and pairing recommendations for MOOD MNKY products.

## Wax Types

Each wax type shows:

- **Name** – e.g., Soy Blend, Coconut Wax
- **Max fragrance load** – Maximum recommended fragrance oil percentage
- **Melt point** – Pouring temperature
- **Description** – Characteristics and best uses

## Wick Options

Wicks are grouped by type:

- **Cotton wicks** – Standard sizes (CD, ECO, etc.)
- **Wood wicks** – Sizes and crackle options

Each wick entry includes:

- **Size** – Diameter or width
- **Recommended container** – Diameter range
- **Wax compatibility** – Which wax types work best

## Pairing Recommendations

Use the reference when:

- Choosing a wick for a container size
- Matching wax to product type
- Determining maximum fragrance load

## Technical Specifications

The MOOD MNKY ecosystem is built using modern technologies:

- **Frontend**: Next.js, TypeScript, Tailwind CSS, ShadCN
- **Backend**: Supabase (PostgreSQL), Serverless Functions
- **E-commerce**: Headless Shopify
- **Integration**: REST APIs, GraphQL, Webhooks
- **AI**: Custom ML models, LLMs, vector databases

For more details on our technology stack, see the [Technology Overview](/reference/architecture/overview).

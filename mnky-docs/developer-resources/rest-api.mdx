## Endpoints

<Tabs>
  <Tab title="User API">
    ### User Management
    Endpoints for managing user accounts, profiles, and preferences.
    
    **Base URL**: `https://api.moodmnky.co/v1/users`
    
    | Endpoint | Method | Description |
    |----------|--------|-------------|
    | `/` | GET | List users |
    | `/:id` | GET | Get user details |
    | `/` | POST | Create user |
    | `/:id` | PUT | Update user |
    | `/:id` | DELETE | Delete user |
  </Tab>
  <Tab title="Products API">
    ### Product Catalog
    Endpoints for accessing product information and inventory.
    
    **Base URL**: `https://api.moodmnky.co/v1/products`
    
    | Endpoint | Method | Description |
    |----------|--------|-------------|
    | `/` | GET | List products |
    | `/:id` | GET | Get product details |
    | `/categories` | GET | List product categories |
    | `/search` | GET | Search products |
  </Tab>
  <Tab title="Orders API">
    ### Order Management
    Endpoints for creating and managing orders.
    
    **Base URL**: `https://api.moodmnky.co/v1/orders`
    
    | Endpoint | Method | Description |
    |----------|--------|-------------|
    | `/` | GET | List orders |
    | `/:id` | GET | Get order details |
    | `/` | POST | Create order |
    | `/:id/status` | PUT | Update order status |
    | `/:id/cancel` | POST | Cancel order |
  </Tab>
</Tabs>

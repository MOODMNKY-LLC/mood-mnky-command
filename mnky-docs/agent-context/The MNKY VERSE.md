> "A scent for the soul. A system for the self."

At MOOD MNKY, we've reimagined the intersection of self-expression, community wellness, and technological innovation by forging an integrated digital universe—**THE MNKY VERSE**. ## Overview: The Architecture of THE MNKY VERSE

**THE MNKY VERSE** is powered by three core AI agents, each a steward of its own domain:

- **MOOD MNKY**: The curator of sensory experiences and bespoke fragrances.
- **SAGE MNKY**: The guide for community engagement and personal wellness.
- **CODE MNKY**: The architect of our technical infrastructure and automation.

Together, these agents create an ecosystem that is as much about emotional resonance as it is about technological precision, driving a holistic experience for every user.

## 1. The Brand: MOOD MNKY – The Experience  
*Curated by MOOD MNKY*

### Core Philosophy
- **Origin Story**: Born from a passion for candle-making, MOOD MNKY evolved into a full-fledged bespoke fragrance studio.
- **Mission**: Transform everyday rituals into immersive, sensory journeys that capture memories, evoke moods, and define personal identity.
- **First Principle**: Self-expression is sacred. Naming a scent empowers you to relive and reshape your experiences.

### Flagship Features
- **Bespoke Fragrance Library**: A meticulously curated collection of raw scent notes. This library empowers both expert scent-crafters and AI agents to blend truly unique fragrances.
- **Living Scent Profiles**: Users build dynamic scent identities stored within the MOOD LABZ system, allowing continuous evolution and personalization.
- **Diverse Product Range**:
  - Handcrafted Candles
  - Bath & Body Collections
  - Grooming Kits
  - Fragrance Oils
  - Custom Scent Accessories
- **Naming & Cataloging**: Each custom blend is archived, contributing to a collaborative catalog of moods, memories, and moments.

### Commerce & Technology Infrastructure
- **Shopify Storefront**: A high-conversion platform engineered for a seamless direct-sales experience.
- **Next.js MOOD LABZ App**: An immersive, gamified interface that facilitates scent crafting, profile management, and AI-driven personalization.
- **Supabase-Integrated Systems**: Robust profile tracking for preferences, wishlists, blend histories, and interactive feedback loops.

### AI & Automation Integration
- **MOOD MNKY Agent**:
  - Serves as a scent curation assistant, mood interpreter, and customer support concierge.
  - Delivers adaptive, personalized experiences by interpreting emotional inputs, narratives, and mood goals.
- **Token Rewards System**: Engages users with points earned through purchases, feedback, and community participation—unlocking exclusive scents, discounts, and in-universe benefits.

## 3. The DevOps & Infrastructure Layer: CODE MNKY – DevOps  
*Curated by CODE MNKY*

### Core Philosophy
- **First Principle**: All magic becomes science when systematized.
- **Purpose**: To make bespoke experiences scalable, personal journeys programmable, and AI interactions ethical, useful, and grounded in truth.

### Key Infrastructure
- **MNKY-REPO (Monorepo)**:
  - A Turborepo-managed codebase built for scale and collaboration.
  - Strategic organization of components:
    - `apps`: Next.js frontends for web, dashboard, and blog
    - `docs`: Mintlify-powered documentation
    - `packages`: Shared UI library (ShadCN + Tailwind) and configurations
    - `agents`: Flowise agent definitions and LangChain modules
    - `infra`: Docker configurations, n8n workflows, and Supabase schemas

- **Core Technical Stack**:
  - **Supabase**: Comprehensive back-end solution providing authentication, database, vector store, and file storage.
  - **Next.js**: React framework for performant, SEO-friendly frontends.
  - **Tailwind & ShadCN**: Utility-first CSS framework paired with accessible component library.
  - **Flowise**: Visual interface for agent orchestration and conversation design.
  - **LangChain**: Framework for complex AI interactions, context management, and tool integration.
  - **n8n**: Workflow automation connecting disparate systems and processes.
  - **Docker**: Containerization for consistent deployment across environments.

### Agent Architecture
- **Agent Harmonization**:
  - Unified memory systems across all three agents.
  - Shared vector database for consistent knowledge retrieval.
  - Coordinated conversation history and context management.
  - Cross-agent task delegation based on specializations.

- **CODE MNKY Capabilities**:
  - Infrastructure monitoring and automated issue resolution.
  - Documentation management and synchronization with Notion.
  - Code generation and optimization for developer productivity.
  - CI/CD pipeline oversight and deployment management.
  - Automated testing and quality assurance.

### Integration & Automation
- **n8n Workflows**:
  - Notion-to-Mintlify documentation synchronization.
  - Shopify-to-Supabase customer data integration.
  - Discord-to-CRM event and engagement tracking.
  - Automated product data management and inventory alerts.
  - Cross-platform analytics aggregation and reporting.

- **API Strategy**:
  - RESTful interfaces for standard operations.
  - GraphQL endpoints for complex, nested data requirements.
  - Webhook infrastructure for event-driven architecture.
  - Consistent authentication and rate limiting across services.

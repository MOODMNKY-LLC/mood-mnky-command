# THE VERSE Pillar: Technical Foundation

The **THE VERSE** pillar is the technological foundation of MOOD MNKY. It is implemented as **MNKY LABZ**—the admin dashboard and command center—plus the underlying technology stack (Next.js, Supabase, Shopify, Vercel).

## Summary

- **MNKY LABZ**: Operational dashboard for formulas, fragrance oils, glossary, Shopify Admin, Notion sync
- **CODE MNKY**: Conversational assistant in the LABZ dock with tools for formulas, oils, glossary
- **Tech Stack**: Next.js 14+, Supabase, Hydrogen React (Storefront API), Vercel

# Introduction to MOOD MNKY Development

Welcome to the MOOD MNKY development ecosystem! This guide provides an overview of our platform architecture, development standards, and key resources to help you get started.

## MOOD MNKY Ecosystem Overview

MOOD MNKY combines physical products, digital experiences, and AI-driven personalization to transform self-care into a meaningful, personalized journey:

```
                  ┌───────────────┐
                  │  Shopify API   │
                  └───────┬───────┘
                          │
                          ▼
┌──────────────┐    ┌───────────────┐    ┌───────────────┐
│ Web Platform │◄───┤ E-commerce   │───►│ Agent System  │
│ (Next.js)    │    │ API Layer     │    │               │
└──────────────┘    └───────────────┘    └───────────────┘
                          │
                          ▼
                  ┌───────────────┐
                  │   Supabase    │
                  │  (Extended    │
                  │    Data)      │
                  └───────────────┘
```

## Repository Organization

Our codebase is organized as a monorepo using Turborepo and pnpm workspaces:

```
MNKY-REPO/
├── apps/                 # User-facing applications
├── packages/             # Shared libraries and utilities
├── agents/               # Agent definitions and configurations
├── content/              # Content libraries and assets
├── docs/                 # Documentation (Mintlify)
├── infra/                # Infrastructure and deployment
├── data/                 # Data models and schemas
├── scripts/              # Utility scripts
├── config/               # Shared configurations
```

## Core Technologies

- **Frontend**: Next.js, TypeScript, Tailwind CSS, ShadCN
- **Backend**: Supabase (PostgreSQL), Serverless Functions
- **E-commerce**: Headless Shopify
- **AI**: Custom agent framework with specialized capabilities
- **Documentation**: Mintlify
- **Build & CI/CD**: Turborepo, GitHub Actions

## Key Development Areas

As a developer, you can contribute to several areas of the MOOD MNKY ecosystem:

### Web Platform

Our web applications use Next.js with the App Router, TypeScript, and a comprehensive UI component library built with Tailwind CSS and ShadCN.

[Web Platform Documentation →](/reference/architecture/web-platform)

### Agent Framework

Our AI agent system consists of three specialized agents - MOOD MNKY, CODE MNKY, and SAGE MNKY - built on a shared infrastructure.

[Agent Framework Documentation →](/agents)

### E-commerce Integration

Our e-commerce system is built around a headless Shopify implementation, integrated with our Next.js frontend and agent system.

[E-commerce Documentation →](/platform/e-commerce)

### Dojo Platform

The Dojo Platform serves as a personalized "life OS" that provides tailored learning and development environments.

[Dojo Platform Documentation →](/platform/dojo)

# MOOD MNKY Guides

Welcome to the MOOD MNKY guides section. Here you'll find comprehensive documentation to help you get the most out of our platform, whether you're a user, administrator, or developer.

## Guide Categories

Our guides are organized into the following categories:

### [Getting Started](/guides/getting-started)

Introductory guides to help new users and developers get up and running quickly with the MOOD MNKY ecosystem.

### [Development Guides](/guides/development)

Technical documentation for developers building on or contributing to the MOOD MNKY platform.

### [User Guides](/guides/user)

Step-by-step instructions for platform users on how to utilize all features and capabilities.

### [Admin Guides](/guides/admin)

Documentation for platform administrators on management, configuration, and maintenance.

## How to Use These Guides

Each guide is designed to be self-contained but may link to other relevant guides or reference documentation. We recommend starting with the Getting Started section if you're new to MOOD MNKY.

## Contributing to the Guides

If you find an issue or would like to suggest improvements to our documentation, please see our [Contribution Guidelines](/guides/development/contribution-guidelines).

# Langchain API Playground

Welcome to the Langchain API Playground. This interactive environment allows you to explore and test the Langchain API directly from your browser.

<Note>
  The API playground is automatically generated from our OpenAPI specification. You can test all endpoints directly from this page.
</Note>

## Available Endpoints

The API playground provides access to all Langchain API endpoints, including:

- **Documents**: Upload, retrieve, and manage documents
- **Embeddings**: Generate embeddings for text
- **Retrieval**: Retrieve relevant documents
- **Chat**: Chat with your documents
- **Agents**: Execute agents with specific capabilities
- **Health**: Check API health

## Authentication

Bearer token authentication is required for Langchain API endpoints.

## Getting Started

To use the API playground:

1. Browse the available endpoints below
2. Click on an endpoint to expand its documentation
3. Fill in the required parameters
4. Click "Try it" to send a request
5. View the response

For more detailed API documentation, see the [Langchain API Reference](/api/devops/langchain/endpoints).

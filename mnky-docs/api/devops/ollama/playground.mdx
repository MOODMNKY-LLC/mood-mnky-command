# Ollama API Playground

Welcome to the Ollama API Playground. This interactive environment allows you to explore and test the Ollama API directly from your browser.

<Note>
  The API playground is automatically generated from our OpenAPI specification. You can test all endpoints directly from this page.
</Note>

## Available Endpoints

The API playground provides access to all Ollama API endpoints, including:

- **Models**: List, create, and delete models
- **Generation**: Generate text completions and chat responses
- **Embeddings**: Generate embeddings for text
- **System**: Get system information and metrics

## Authentication

No authentication is required for the Ollama API.

## Getting Started

To use the API playground:

1. Browse the available endpoints below
2. Click on an endpoint to expand its documentation
3. Fill in the required parameters
4. Click "Try it" to send a request
5. View the response

For more detailed API documentation, see the [Ollama API Reference](/api/devops/ollama/endpoints).

# n8n API Playground

Welcome to the n8n API Playground. This interactive environment allows you to explore and test the n8n API directly from your browser.

<Note>
  The API playground is automatically generated from our OpenAPI specification. You can test all endpoints directly from this page.
</Note>

## Available Endpoints

The API playground provides access to all n8n API endpoints, including:

- **Workflows**: Create, update, activate, and manage workflows
- **Executions**: Execute workflows and get execution results
- **Credentials**: Manage credentials for integrations
- **Health**: Check API health

## Authentication

Bearer token authentication is required for n8n API endpoints.

## Getting Started

To use the API playground:

1. Browse the available endpoints below
2. Click on an endpoint to expand its documentation
3. Fill in the required parameters
4. Click "Try it" to send a request
5. View the response

For more detailed API documentation, see the [n8n API Reference](/api/devops/n8n/endpoints).

# Flowise API Playground

Welcome to the Flowise API Playground. This interactive environment allows you to explore and test the Flowise API directly from your browser.

<Note>
  The API playground is automatically generated from our OpenAPI specification. You can test all endpoints directly from this page.
</Note>

## Available Endpoints

The API playground provides access to all Flowise API endpoints, including:

- **Chatflows**: Create, update, and manage chatflows
- **Predictions**: Execute chatflow predictions
- **Components**: Get component information
- **Credentials**: Manage credentials
- **Tools**: Access available tools
- **Health**: Check API health

## Authentication

Bearer token authentication is required for most Flowise API endpoints.

## Getting Started

To use the API playground:

1. Browse the available endpoints below
2. Click on an endpoint to expand its documentation
3. Fill in the required parameters
4. Click "Try it" to send a request
5. View the response

For more detailed API documentation, see the [Flowise API Reference](/api/devops/flowise/endpoints).

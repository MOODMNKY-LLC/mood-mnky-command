# Bungie API Playground

Welcome to the Bungie API Playground. This interactive environment allows you to explore and test the Bungie API directly from your browser.

<Note>
  The API playground is automatically generated from our OpenAPI specification. You can test all endpoints directly from this page.
</Note>

## Available Endpoints

The API playground provides access to all Bungie API endpoints, including:

- **App**: Get API usage and application information
- **User**: Access user profile information
- **Destiny2**: Access Destiny 2 game data
- **GroupV2**: Access clan and group information

## Authentication

Bearer token authentication is required for Bungie API endpoints.

## Getting Started

To use the API playground:

1. Browse the available endpoints below
2. Click on an endpoint to expand its documentation
3. Fill in the required parameters
4. Click "Try it" to send a request
5. View the response

For more detailed API documentation, see the [Bungie API Reference](/api/devops/bungie/endpoints).
